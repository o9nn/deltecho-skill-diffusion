# Deep Tree Echo - Skill Diffusion Preview

🌳 **Interactive preview of the Deep Tree Echo cognitive architecture with Skill-Diffusion enhancements.**

## Live Demo

Visit the live demo at: **https://o9nn.github.io/deltecho-skill-diffusion/**

## Features Showcased

### 🧠 Cognitive Architecture
- **Double Membrane System**: Outer (objective) and inner (subjective) membranes with transjective buffer
- **Sys6 Operadic Composition**: Mathematical foundation with 30-step cycle and 42 sync events
- **Hypergraph Memory**: Declarative, procedural, episodic, and intentional memory spaces

### 🎯 Trajectory Diffusion
- **DDPM-based Generation**: 1000-step diffusion process for trajectory proposals
- **Intent-Conditioned Sampling**: Generate trajectories based on specified goals
- **Latent Space Encoding**: Efficient representation of complex trajectories

### ⚔️ Opponent Processing
- **4-Phase Cycle**: Propose → Decode → Normalize → Commit
- **Free Energy Minimization**: Automatic refinement toward optimal solutions
- **Skill Learning**: Autonomous acquisition and storage of reusable patterns

### ✨ Gesture Glyph Codec
- **4 Glyph Formats**: Stroke, TimeChannel, VectorField, ContactMap
- **Probabilistic Decoding**: Multiple hypothesis generation with confidence scoring
- **Noise-Tolerant Reconstruction**: Robust trajectory recovery from visual representations
